Transitive Dependency Highlights

Notable transitive dependencies brought in by direct deps:

libsqlite3-sys (via rusqlite) BUILD DEP
Compiles SQLite from C source at build time. Uses cc crate for compilation. This is the standard way to bundle SQLite in Rust. The SQLite source is vendored and well-audited. Build-time only dependency — no runtime network activity.
proc-macro2, syn, quote (via clap, serde) COMPILE ONLY
Standard Rust procedural macro infrastructure. Used by #[derive(...)] macros for Clap (CLI parsing) and Serde (serialization). These only run at compile time and produce no runtime code beyond the derived implementations.
aho-corasick, memchr (via regex) SAFE
Fast string searching algorithms used by the regex engine. Pure computational libraries with no I/O, no system calls, no side effects. Written by the same author (Andrew Gallant / BurntSushi).
